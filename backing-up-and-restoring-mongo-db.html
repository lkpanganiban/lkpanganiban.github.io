Backing up and Restoring Mongo DB

Mongo DB is a type NoSQL database which treats your data like JSON- documents (key-value pair). In this post, I will be showing some steps in backin g up and restoring a Mongo DB.

We will use Mongo DB's built-in utility to backup and restore data.

Features

  • Dumping Mongo DB data
mongodump --db mongo-db-name --collection mongo-collection-name --out - | gzip >  mongo-collection-name-backup.gz
  • Restoring Mongo DB data
gunzip -c mongo-collection-name.gz > mongo-collection-name-backup.bson
mongorestore --db mongo-db-name --collection mongo-collection-name mongo-collection-name-backup.bson
