Development Machine

Setup and Requirements

  1. python version 2.7

    python -v

  2. Installing Python Package Manager

    [sudo] apt-get install python-pip

  3. Installing Virtual Environment

    [sudo] pip install --proxy http://proxy:port virtualenv

  4. Creating and Activating Virtual Environment

    virtualenv foss4gEnv

    source foss4gEnv/bin/activate

  5. Installing Flask

    (foss4gEnv)pip install --proxy http://proxy:port flask
